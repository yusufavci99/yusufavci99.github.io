CRSG Author Module

# Aug 5, 2021

Code Review Serious Game Author Module was my summer 2020 internship project. I created a serious game used in teaching the code review process. Players should fix the reviewed codes. Players’ submissions are evaluated and scored.

Used JS, Ace Editor Library, Judge0 API, Ajax, Firebase.

There are 3 levels to select.
