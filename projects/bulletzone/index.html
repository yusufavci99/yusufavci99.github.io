Bullet Zone

# Sep 10, 2021

APK File

A bullet hell style casual mobile game created in Unity. The game is 3D and the projectiles come both horizontally and vertically. Yellow points show the points where the player can go. Each dodged projectile increases your score. The game was done in 3 hours. I am planning to update the game and put it on the Play Store.
